关于css的一些工具

### 关于其它的一些移动端自适应布局方案
```css
html {
    font-size:20px
}
@media only screen and (min-width:400px) {
    html {
        font-size:21.33333333px!important
    }
}
@media only screen and (min-width:414px) {
    html {
        font-size:22.08px!important
    }
}
@media only screen and (min-width:480px) {
    html {
        font-size:25.6px!important
    }
}
```
来自于平安银行
